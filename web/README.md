# Web Project

## Description
This project is a Next.js application that serves as the frontend for a bidding system, focusing on a platform for buying and selling items, likely with a focus on cats. It interacts with the backend services provided by the Bidding and Payment projects. It uses WebSockets for real-time updates and provides a user interface for browsing items, placing bids, and processing payments. The landing page features a hero section, a features section, a spotlight cat section, and a featured listings section.

## How to Run
To run the Next.js frontend:
1.  Make sure you have Node.js and npm installed.
2.  Navigate to the `web` directory.
3.  Install the dependencies: `npm install`
4.  Run the development server: `npm run dev`
5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Details
-   The application is built with Next.js.
-   It uses WebSockets for real-time communication with the backend.
-   It provides a user interface for browsing items, placing bids, and processing payments.
-   The landing page is located at `app/(landing)/page.tsx`.
